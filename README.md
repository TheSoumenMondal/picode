**PiCode** is a stylish, easy-to-use web app that allows developers to generate and share stunning code snippets in seconds. Designed with simplicity and aesthetics in mind, PiCode is perfect for sharing code on social media, blogs, or with teammates.

![Image](https://github.com/user-attachments/assets/f737d694-ab99-43e4-b65e-cf43e3e0b06f)


## Features

- Beautiful code snippet preview with custom themes
- Light/Dark mode toggle (via navbar)
- One-click export/download of snippets
- Share-ready output for social platforms

---

## Tech Stack

- **Next.js**
- **TypeScript**
- **Tailwind CSS** + **shadcn/ui**
- **Framer Motion**
- **Tabler Icons**
- **Zustand**

---

##  Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/TheSoumenMondal/picode.git
cd picode
npm install
```
### Run locally
```bash
npm run dev
```
